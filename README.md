# 新能源汽车资讯平台系统 python1102

## 项目概述

新能源汽车资讯平台系统是一个基于Python技术的B/S架构信息系统，旨在提供新能源汽车相关的新闻、数据、分析等信息。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员：负责系统的维护与管理
- 用户：浏览和使用系统功能的主体

## 功能模块

系统主要包括但不限于以下功能模块：

- 新闻资讯展示
- 数据分析与报告
- 用户互动平台

## 系统特性

- **前后端分离**：系统采用前后端分离的架构，前端使用Vue框架。
- **丰富的文档支持**：提供详细的项目文档，包含多种图表模板。

### 文档与模板

- 包含用例图、ER图、结构图、实体属性图、时序图、流程图等Visio模板。
- 提供图文修改教程资料，便于用户定制化系统。

### 开发与调试支持

- **开发工具提供**：提供项目所需的全部软件。
- **视频教程**：提供视频调试教程及同类技术开发项目系统核心技术讲解视频。
- **代码位置指导**：明确主要代码位置，帮助快速定位与功能实现。

## 运行环境

- 服务器环境：建议使用Python支持的服务器环境。
- 客户端环境：现代浏览器。

1. 配置服务器环境。
2. 部署后端服务。
3. 部署前端服务。
